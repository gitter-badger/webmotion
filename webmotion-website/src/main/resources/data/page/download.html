Maven Utiliser maven pour récupérer WebMotion

Un repository maven 2 est disponible en ajoutant les lignes suivantes dans votre pom.xml de votre projet :

<repositories>
    <repository>
        <id>debux.release</id>
        <name>DebuxReleaseRepository</name>
        <url>http://debux.org/maven/repo</url>
        <releases>
            <enabled>true</enabled>
            <checksumPolicy>fail</checksumPolicy>
        </releases>
    </repository>
</repositories>
...
<dependency>
    <groupId>org.debux.webmotion</groupId>
    <artifactId>webmotion</artifactId>
    <version>2.0</version>
</dependency>

Sources Récupérer les sources

Les sources sont disponibles sous subversion, vous pouvez les récupérer en accès anonyme en exécutant la commande suivante :

$ svn checkout http://svn.debux.org/webmotion/trunk/
