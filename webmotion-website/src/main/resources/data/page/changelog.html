Changelog

Cette partie regroupe l'ensemble des modifications entre les différentes versions de WebMotion.

Sur chaque version vous retrouvez :

  • les modifications majeurs
  • l'ensemble des tickets du redmine réalisés
  • l'ancienne version du site web avec la documentation

Version 2.3

Major changes:

  • Une ressource est détecté static si le fichier exiiste
  • Simplification de la configuration des extras
  • Renommage de CookieManger en CookieManager
  • Support des WebSocket
  • Extra Apache Shiro
  • Support de GSP, Sacalate, ZK, ...
  • Plugin pour NetBeans
  • Modification des méthodes de la classe ServerListener

Tickets:

  • Bug #268: Le projet webmotion-test ne build pas sous Windows
  • Bug #280: ExecutorParametersInjectorHandler contient plusieurs fois le même injecteur
  • Feature #121: Creation d'un extra avec Apache Shiro pour gérer la sécurité
  • Feature #124: Plugin pour NetBeans
  • Feature #235: Support des Websockets
  • Feature #252: Revue des logs
  • Feature #253: Multiple listeners du serveur
  • Feature #261: Vérification des actions java
