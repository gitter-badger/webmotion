Wikimotion

Présentation

WikiMotion est un moteur de wiki, qui peut à la fois être utilisé en standalone ou comme extension. Par exemple, le site web WebMotion utilise le wiki en tant qu'extension.

Il propose comme fonctionnalités :

  • Sauvegarde de fichiers plats (pas besoin de base de données)
  • Syntaxe des pages en HTML, RST ou LaTeX
  • Support de l'authentification HTTP basic
  • Upload de fichier
  • Site map
  • Media map

Le style du wiki repose sur le boostrap de Twiter. Vous pouvez tout de même réaliser votre propre layout en redéfinissant le fichier layout.jsp.

Installation

Tout d'abord veuillez récupérer la dernière version de WikiMotion disponible dans la section Téléchargement du site. Puis déployer le war téléchargé dans un serveur d'application comme Tomcat, Jetty ou encore GlassFish.

Une fois le wiki déployer, vous pouvez modifier le fichier de configuration wikimotion-config.properties fourni par défaut, afin de configurer et personnaliser votre wiki.

Si vous souhaitez le support du Latex dans le wiki, il vous faudra installer latex ainsi que tth. 

$ apt-get install latex tth

Si vous souhaitez le support du RST dans le wiki, il vous faudra installer docutils ainsi que le script rst2html-pygments. 

$ apt-get install python-docutils python-pygments

$ wget http://docutils.sourceforge.net/sandbox/code-block-directive/tools/pygments-enhanced-front-ends/rst2html-pygments
$ cp rst2html-pygments /usr/bin/rst2html-pygments
$ chmod 755 /usr/bin/rst2html-pygments

Les utilisateurs sont configurés dans le fichier wikimotion-users.json, il exsite un utilisateur admin avec comme mot de passe admin par défaut. Les mots de passe doivent être encodés en sha1.

Extension

Pour pouvoir utiliser le WikiMotion en extension, il faut rajouter dans vos dépendances de votre projet le wiki. 

<dependency>
    <groupId>org.debux.webmotion</groupId>
    <artifactId>wikimotion</artifactId>
    <version>2.2</version>
    <type>war</type>
    <scope>runtime</scope>
</dependency>

Ensuite vous pouvez directement l'utiliser dans votre mapping en tant qu'extension. 

[extensions]
/wiki                                             wikimotion.wm
